发表于 2022-04-02 更新于 2022-04-04 分类于 java

Bean实例化过程

  • 工厂类通过构造器行参传递要解析的xml配置文件
  • 解析配置文件，得到对应Bean标签的id和class属性值，生成MyBean对象集合
  • 实例化Bean对象，便利MyBean对象集合，通过class全类名反射实例，将实例保存到hashmap中
  • getBean方法，通过id获取从hashmap中获取实例化的Bean对象
